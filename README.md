## Atividades
1. **Crie um programa que calcule a soma de dois números digitados pelo
usuário.**
--------------------------
#include <stdio.h>
 
 int main() {
 
    int x,y;
    
    printf("Insira o primeiro numero : \n");
    
    scanf("%d", &x);
    
    printf("Insira o segundo numero: \n");
    
    scanf("%d", &y);
    
    printf("Soma: %d", x+y);
    
    return 0;
}
